# Peta Rumah Sakit Bandung

Proyek interaktif Leaflet.js dengan data rumah sakit Bandung (Kabupaten & Kota).

## Fitur
- Peta interaktif dengan tema dark
- Sidebar untuk pencarian & filter
- Popup info (nama, alamat, kategori)
- Rute tercepat langsung di peta
- Mode arah: Mobil (default)
- Deteksi lokasi otomatis
- Identitas kelompok HKI

## Cara Menjalankan
1. Ekstrak folder `rumahsakit-bandung`
2. Buka `index.html` di browser (disarankan Chrome)
3. Izinkan akses lokasi jika diminta
4. Klik marker → “Lihat Arah” untuk melihat rute

## Catatan
- Saat offline, peta tetap berfungsi menggunakan cache browser.
- Jika online, sistem otomatis gunakan data dan rute real-time.
